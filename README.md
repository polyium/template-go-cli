## Getting Started

### Installing

```bash
brew tap polyium/homebrew-taps

brew install template-go-cli
```

###### Upgrading

```bash
brew update && brew upgrade template-go-cli
```

###### Reinstalling

```bash
brew update
brew uninstall --cask template-go-cli
brew install template-go-cli
```

## Usage

```bash
template-go-cli --help
```
